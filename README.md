# Sint
Minimal syn-ack scanner.

## Usage
```shell
Usage: scan [OPTIONS] --output <OUTPUT> --port <PORT>

Options:
  -o, --output <OUTPUT>        Output file
  -p, --port <PORT>            Port to be scanned
  -i, --interface <INTERFACE>  Interface to scan on
```
